# doumi (dumi)

> A very simple text-format based on a [deadfish](https://esolangs.org/wiki/Deadfish) interpreter

Ops Supported:
---

| Op | Description |
|----|-------------|
| i  | Incement    |
| d  | Decrement   |
| o  | Output      |
| s  | Square      |

Other Features:

- single-line comments
Example:
```deadfish
# A comment
ii
o # should print 2
```
